# 💊 의약품 정보 챗봇 프로젝트

## 프로젝트 소개 및 목표
식품의약품안전처 공공데이터를 기반으로, 사용자가 증세 및 약품명·효능·사용법·부작용 등 주요 정보를 간편하게 질의할 수 있도록 답변하는 RAG(검색+생성) 챗봇을 개발합니다. 

---

## 팀원 소개 및 역할 분담
| 이름   | 역할           | 주요 담당                                    |
| ------ | -------------- | -------------------------------------------- |
| 채서린 | 팀장    | 전체 시스템 설계, RAG 파이프라인 및 챗봇 구현   |
| 천현진 | 데이터/백엔드   | 약품 리스트 파싱, 벡터DB, API 구성           |

| 황혜윤 | 프론트엔드/UI  | 챗봇 CLI/웹 인터페이스 개발                   |
| 정지아 | 문서화/테스트   | 프로젝트 문서화, QA, 발표자료                  |
| 염한결 | 무사귀가   | 엄한 업체에 발 묶이지 말고 무사히 돌아오기          |

---

## 기술 스택
- Python 3.8+
- LangChain, OpenAI GPT-4o API
- FAISS (벡터 데이터베이스)
- Streamlit
- pip, Git, VS Code

---

## 주요 기능 설명
- **약품명/효능/사용법/부작용 질의응답**: 다양한 약품 정보를 자동으로 안내
- **유사도 기반 검색 및 요약**: NLP 임베딩을 이용해 가장 관련성이 높은 답변 제공
- **한국어 지원 인터페이스**: CLI 또는 웹 기반으로 누구나 이용
- **정확한 데이터 근거 답변**: 공식 데이터만 기반, 무책임 생성 응답 방지
